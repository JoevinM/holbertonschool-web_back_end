## Project Overview

This project introduces **ES6 Classes**, a modern way to structure and organize JavaScript code using object-oriented programming principles.

You will learn to:

- Define and use ES6 classes
- Implement constructors, methods, and static methods
- Use getters and setters
- Apply inheritance and polymorphism
- Work with abstract-like classes
- Override methods in child classes
- Use symbols and metaprogramming basics
- Understand hoisting issues with classes

This project is part of **Foundations v2.1 â€“ Part 3**.

## Learning Objectives

By the end of this project, you should be able to explain, without external help:

### General
- How to define a Class in JavaScript
- How to add methods to a class
- Why and how to use static methods
- How to extend a class from another
- How inheritance works in ES6
- How to override methods
- What Symbols are used for
- The basics of metaprogramming
- How JavaScript class hoisting works
- How implicit type conversion works using `Symbol.toPrimitive`

## Requirements

### General
- All files will be interpreted on **Ubuntu 20.04 LTS**
- Using **NodeJS 20.x.x** and **npm 9.x.x**
- Allowed editors: `vi`, `vim`, `emacs`, `Visual Studio Code`
- All files must end with a new line
- A `README.md` file at the root of the project is mandatory
- All files must use the `.js` extension
- All functions and classes must be exported
- Code will be tested using **Jest**
- Code will be analyzed using **ESLint**
- Code must pass all tests and lint checks
- You can run the full test suite with `npm run full-test`
