## Project Overview

This project introduces **ECMAScript 6 (ES6)**, the modern version of JavaScript that improves syntax, readability, and developer productivity.

You will learn to:

- Understand what ES6 is and why it exists
- Use modern variable declarations (`const`, `let`)
- Write cleaner functions with arrow syntax
- Manipulate data with rest and spread operators
- Use template literals for string interpolation
- Improve object creation and manipulation
- Work with loops, iterators, and structured objects

This project is part of **Foundations v2.1**.

---

## Learning Objectives

By the end of this project, you should be able to explain, without external help:

### General
- What **ES6** is
- New features introduced in ES6
- The difference between `var`, `let`, and `const`
- Block-scoped variables
- Arrow functions
- Default function parameters
- Rest and spread operators
- Template literals
- Object property shorthand
- Computed property names
- ES6 method properties
- Iterators and `for...of` loops

## Requirements

### General
- All files will be interpreted on **Ubuntu 20.04 LTS**
- Using **NodeJS 20.x.x** and **npm 9.x.x**
- Allowed editors: `vi`, `vim`, `emacs`, `Visual Studio Code`
- All files must end with a new line
- A `README.md` file at the root of the project is mandatory
- All files must use the `.js` extension
- All functions must be exported
- Code will be tested using **Jest**
- Code will be analyzed using **ESLint** with provided rules
